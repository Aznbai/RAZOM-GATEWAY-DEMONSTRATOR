Razom: Audible Motion

Razom is a wearable sensor kit offering motion control for MIDI equipment, developed and produced by Rost Likholat.

A Razom kit includes multiple sender nodes and a wireless receiver which is a MIDI gateway.

Each sender node operates as a multidimensional theremin, detecting its own motion rather than that of its surroundings. The gateway offers a web-based interface you are currently looking at.

It is used for mapping and managing the collected data, which is subsequently converted into real-time MIDI control parameters.

If not licensed otherwise all parts of this project are public domain.
